# RTI-Transport - Fabric Workspace

This repository is synced with the **RTI-Transport** Microsoft Fabric workspace via Git Integration.

Fabric items (Eventhouses, Eventstreams, Notebooks, Dashboards, Reports, Data Agents) are managed in the root directory.

## Deploy Scripts (Legacy)

The original deployment scripts that created the workspace items via Fabric REST API are preserved in the [`_deploy/`](_deploy/) folder.
